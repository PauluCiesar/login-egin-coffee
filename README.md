## ☕ Tela de Login — Egin Coffee

Esta tela de login foi projetada para transmitir calma, conforto e confiança, refletindo a identidade acolhedora de uma cafeteria. O uso do verde oliva nos botões e campos de entrada remete à natureza e segurança, enquanto o fundo claro proporciona leveza visual e reduz o cansaço dos olhos. A tipografia suave, os elementos arredondados e a hierarquia bem definida guiam o usuário de forma intuitiva até a ação principal: entrar. O resultado é uma experiência simples e agradável.
